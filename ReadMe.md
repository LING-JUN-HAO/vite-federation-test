# vite-federation-test

## 安裝方式

```bash
npm install
```
## 啟動方式

【 host 端 】

啟動伺服器預設使用：http://localhost:5000

```bash
npm run preview
```

**每次更新均需要執行`npm run build`**

【 Remote 端 】

啟動伺服器預設使用：http://localhost:5173

```bash
npm run dev
```
